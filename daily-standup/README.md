# Daily Standup

This script creates a daily standup post on weekdays at 9:30pm UTC and tags members of the team.

### Prerequisites

Add your Campsite API key to a .env file:

```
// .env
CAMPSITE_API_KEY=<YOUR_API_KEY>
```

### Running the script

```
// Install dependencies
pnpm i

// Run the script
pnpm start
```
